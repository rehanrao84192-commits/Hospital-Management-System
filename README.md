# Hospital Management System

## 📌 Project Description
Hospital Management System ek Python + MySQL based project hai jo hospital ki management ko easy banata hai.

Is project me:
- Doctors add kar sakte hain
- Patients manage kar sakte hain
- Appointments book kar sakte hain
- Doctors list dekh sakte hain

---

## 🛠 Technologies Used
- Python
- Tkinter GUI
- MySQL Database
- mysql-connector-python

---

## 💻 Features
- Add Doctor
- Add Patient
- Book Appointment
- View Doctors
- Database Connectivity
